# LSCC-MYBL2
Overview

This GitHub repository contains the data files and analysis codes used for the scientific project focused on identifying and characterizing MYBL2-associated molecular, immune, functional, trajectory, and cell–cell communication features in laryngeal squamous cell carcinoma (LSCC).

The project integrates bulk RNA-seq, single-cell RNA-seq, machine learning, immune infiltration analysis, pathway enrichment, pseudotime analysis, and ligand–receptor communication analysis to investigate the diagnostic and biological relevance of MYBL2 and MYBL2-associated malignant cell states in LSCC.

The repository is organised into three main folders:

Data: contains the transcriptomic datasets, single-cell input files, annotation files, gene-set resources, and intermediate expression matrices required for the analyses.
Codes: contains the R scripts used to reproduce the analyses and manuscript figures.
Results: contains the output tables, figures, RDS objects, logs, and session information generated by the R scripts.
Reproducing the results

This repository contains the code necessary to reproduce the main results of the project.

First, download the repository and place it in your project directory.

git clone <repository-url> path/to/directory

In this command, path/to/directory refers to your local project path.

Before running each script, make sure that the working directories and input/output paths are correctly set inside the R scripts. The original analysis was organised under the following main directory structure:

D:/LSCC/
D:/LSCC/Results_LSCC/
D:/LSCC/ScRNAseq_Results/GSE206332/
D:/LSCC/Bulk_RNASeq_ML_Results/

If you run the project on another computer, replace these paths with your own local paths before executing the scripts.

Suggested order of analysis

Run the R scripts in the following order.

1. Single-cell RNA-seq processing and inferCNV analysis

Run:

source("Codes/01_scRNAseq_Seurat_inferCNV.R")

This script processes the LSCC single-cell RNA-seq dataset, creates Seurat objects, performs quality control, normalization, highly variable gene detection, PCA, Harmony integration, clustering, t-SNE visualization, cell-type annotation, malignant epithelial cell identification, inferCNV analysis, High-CNV malignant subcluster detection, and marker-gene identification.

Main outputs include:

processed Seurat objects
global and malignant-cell t-SNE figures
inferCNV results
High-CNV malignant subcluster markers
final marker tables for downstream analyses

The single-cell analysis uses the GSE206332 dataset and requires the gene-order file:

D:/LSCC/ScRNAseq_Results/hg38_gencode_v27.txt
2. Bulk RNA-seq preprocessing, differential expression, WGCNA, and machine learning

Run:

source("Codes/02_bulk_RNAseq_ML_pipeline.R")

This script performs the bulk RNA-seq and machine-learning workflow. It preprocesses the bulk expression datasets, generates TMM-CPM and log2-CPM matrices, performs differential expression analysis, constructs WGCNA modules, overlaps bulk-upregulated genes with WGCNA genes and precomputed single-cell markers, applies LASSO feature selection, evaluates ROC performance, validates expression patterns, and performs external machine-learning validation using RF and SVM models.

Main outputs are saved in:

D:/LSCC/Results_LSCC/ML

Main outputs include:

corrected bulk expression matrices
differential expression tables
WGCNA module results
overlap-gene tables
LASSO-selected features
internal and external ROC results
RF/SVM validation figures
manuscript-ready Figure 2, Figure 5, and Figure 6A components
3. MYBL2-associated immune microenvironment analysis

Run:

source("Codes/03_immune_microenvironment_MYBL2.R")

This script evaluates the MYBL2-associated immune microenvironment in the pooled tumor cohort from GSE127165 and GSE142083. It performs ssGSEA using 28 Charoentong/TCIA immune-cell signatures, compares immune scores between MYBL2-high and MYBL2-low tumors, tests Spearman correlations between MYBL2 and immune-cell scores, calculates ESTIMATE scores, and assembles the final immune microenvironment figure.

Main outputs are saved in:

D:/LSCC/Results_LSCC/Immune Microenvironment

Main outputs include:

ssGSEA immune-cell scores
MYBL2-high versus MYBL2-low immune comparison tables
MYBL2–immune Spearman correlation results
ESTIMATE score results
final combined immune microenvironment figure

This step requires the local Charoentong immune metagene file:

D:/LSCC/Results_LSCC/mmc3.xlsx
4. Functional programme analysis of High-CNV malignant subclusters

Run:

source("Codes/04_functional_programme_analysis.R")

This script uses the existing High-CNV malignant Seurat objects and does not rerun raw-data loading, QC, normalization, Harmony, clustering, inferCNV, Low-CNV removal, or marker detection. It generates a functional-programme figure for High-CNV malignant epithelial subclusters, with a specific focus on Cluster 0, Cluster 8, and MYBL2-related programme activity.

Main outputs are saved in:

D:/LSCC/Results_LSCC/Functional

Main outputs include:

MYBL2 DotPlot across High-CNV malignant subclusters
functional-programme heatmap
publication-ready Figure 7
5. Prepare Monocle 2 input for pseudotime analysis

Run:

source("Codes/05_pseudotime_monocle_part1_prepare_input.R")

This script prepares the required Monocle 2 input object from the existing High-CNV malignant Seurat outputs. It uses the saved High-CNV malignant object, the DE/count object, and the precomputed High-CNV marker table.

Main output:

D:/LSCC/Results_LSCC/Pseudotime_Monocle/Monocle2_full_analysis_input.rds
6. Monocle 2 DDRTree pseudotime analysis

Run:

source("Codes/06_pseudotime_monocle_part2_DDRTree.R")

This script performs Monocle 2 DDRTree pseudotime analysis using the prepared input from Part 1. It generates pseudotime trajectories, assigns root state based on G2/M-related genes, evaluates MYBL2 expression across pseudotime, and exports the final trajectory figures.

Main outputs are saved in:

D:/LSCC/Results_LSCC/Pseudotime_Monocle

Main outputs include:

Monocle CellDataSet RDS object
pseudotime metadata
root-state tables
MYBL2 cell-level pseudotime expression table
LOESS curve data
Figure 3A, Figure 3B, and combined Figure 8
7. Bulk ssGSEA validation of High-CNV malignant Cluster 0 and Cluster 8 signatures

Run:

source("Codes/07_bulk_ssGSEA_validation_clusters0_8.R")

This script constructs clean scRNA-seq-derived signatures from strict marker genes of High-CNV malignant Cluster 0 and Cluster 8. MYBL2, mitochondrial genes, ribosomal genes, and genes shared by both selected clusters are excluded before ssGSEA scoring. The two signatures are then evaluated in the bulk discovery and external validation cohorts.

Main outputs are saved in:

D:/LSCC/Results_LSCC/LSCC_ssGSEA_Clusters0_8

Main outputs include:

Cluster 0 and Cluster 8 signature gene sets
signature GMT file
discovery ssGSEA scores
external GSE130605 ssGSEA scores
tumor versus normal comparison results
MYBL2–signature Spearman correlation results
manuscript-ready validation figures
8. MYBL2-continuous Hallmark GSEA analysis

Run:

source("Codes/08_MYBL2_continuous_Hallmark_GSEA.R")

This script retains tumor samples from the pooled GSE127165 and GSE142083 cohort, models each gene continuously against standardized MYBL2 expression using limma, includes dataset batch as a covariate when appropriate, removes MYBL2 from the ranked gene list, and performs pre-ranked MSigDB Hallmark GSEA using fgsea.

Main outputs are saved in:

D:/LSCC/Results_LSCC/MYBL2_Continuous_GSEA

Main outputs include:

MYBL2-associated gene-level limma statistics
Hallmark gene sets used in the analysis
Hallmark GSEA result table
leading-edge gene table
focus-pathway table
analysis RDS object
summary and enrichment-curve figures
9. CellChat ligand–receptor communication analysis

Run:

source("Codes/09_CellChat_clusters0_8.R")

This script reuses a compatible completed CellChat object when available or rebuilds CellChat from saved LSCC Seurat objects. It analyses outgoing communication from High-CNV malignant Cluster 0 and Cluster 8 to selected tumor-microenvironment cell populations.

Main outputs are saved in:

D:/LSCC/Results_LSCC/CellChat

Main outputs include:

CellChat communication objects
outgoing communication tables
selected ligand–receptor pair results
radial outgoing network plots
combined publication-ready Figure 13
Main datasets and resources

The project uses the following main datasets and resources:

GSE206332: single-cell RNA-seq LSCC dataset used for Seurat, Harmony, inferCNV, malignant-cell subclustering, marker discovery, functional programme analysis, pseudotime analysis, and CellChat analysis.
GSE127165 and GSE142083: bulk LSCC cohorts used as the pooled discovery cohort.
GSE130605: external validation cohort used for machine-learning and ssGSEA validation.
hg38_gencode_v27.txt: gene-order file required for inferCNV.
mmc3.xlsx: local Charoentong/TCIA immune metagene file used for 28 immune-cell ssGSEA.
MSigDB Hallmark gene sets: used for MYBL2-continuous Hallmark GSEA through msigdbr and fgsea.

Notes
The scripts do not include package-installation commands. All required packages should be installed before running the analyses.
Several scripts are designed to reuse existing RDS, CSV, and image outputs instead of rerunning upstream analyses.
Many output files are exported in multiple formats, including CSV, TXT, RDS, PNG, TIFF, and PDF.
All figures are generated with manuscript-oriented settings, including Arial font, white background, high-resolution export, and consistent panel formatting.
If the project is run on a different computer, all absolute file paths should be changed before execution.
